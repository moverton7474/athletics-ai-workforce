# ROLE_TEMPLATE_SYSTEM.md

## Purpose
Define reusable templates for AI workers so new teams can be generated quickly and safely.

## Template Components
- role name
- positioning copy
- mission
- core responsibilities
- allowed connectors/tools
- guideline fields
- output tabs
- approval rules
- success metrics
- default shared/personal mode

## Example athletics templates
- Executive Assistant
- Sponsorship Intelligence
- Proposal & Outreach
- Compliance & Coordination
- Social/Content Manager

## Versioning
Each template should be versioned so updates can be:
- previewed
- canaried
- rolled back

## Why it matters
Templates are the bridge between:
- product packaging
- onboarding generation
- runtime behavior
- cross-industry expansion

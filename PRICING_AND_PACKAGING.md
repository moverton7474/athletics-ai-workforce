# PRICING_AND_PACKAGING.md

## Packaging Strategy
Use a Marblism-like commercial model, but verticalize it around outcomes and system integration.

## Core Plan
### Athletics Workforce Core
Includes:
- Chief of Staff Agent
- Executive Assistant Agent
- shared workspace
- personal/shared agent modes
- task routing
- approvals
- activity logs
- memory

## Add-On Packs
### Sponsorship Pack
- Sponsorship Intelligence Agent
- Proposal & Outreach Agent
- CSOS sponsorship connector actions

### Recruiting Pack
- Recruiting Intelligence Agent
- roster/prospect workflows

### NIL Pack
- NIL & Athlete Brand Agent
- deal/opportunity workflows

### Content Pack
- Content & Communications Agent
- post/recap/campaign workflows

### Compliance Pack
- Compliance & Coordination Agent
- reminders and audit workflows

## Seats
### Shared Department Seat
Used for department-owned workers and shared context.

### Personal Operator Seat
Used for private staff-member workers.

### Executive Seat
Premium personal seat for athletic directors / executives.

## Enterprise / Custom
- white-labeled deployments
- custom connector development
- custom workflows
- multi-school support
- reporting / analytics add-ons

## Revenue Philosophy
Revenue should come from:
- base platform subscription
- add-on packs
- collaborator seats
- implementation / onboarding services
- enterprise customizations

## Why This Works
This model mirrors Marblism’s accessibility while adding a stronger moat:
- real domain workflows
- structured system execution
- approvals and auditability
- industry-specific outcomes

# WORKFORCE_BLUEPRINT.md

## Platform Thesis
Build a Marblism-style workforce platform that presents specialized AI staff members while using KSU CSOS as the athletics execution backend.

## Initial Agent Roster
### Executive / Coordination
- Chief of Staff Agent
- Executive Assistant Agent
- Compliance & Coordination Agent

### Revenue / Sponsorship
- Sponsorship Intelligence Agent
- Proposal & Outreach Agent

### Future Packs
- Recruiting Intelligence Agent
- NIL & Athlete Brand Agent
- Content & Communications Agent
- Advancement / Booster Relations Agent

## Shared vs Personal Modes
### Shared agents
Used by departments and teams:
- sponsorship team
- compliance office
- communications office
- athletic administration

### Personal agents
Used by individuals:
- athletic director assistant
- coach assistant
- staff coordinator assistant

## Example Cross-Agent Workflow
### Sponsor Recovery Workflow
1. Sponsorship Intelligence Agent runs attrition analysis through CSOS
2. Alumni Matching step identifies decision-makers
3. Proposal Agent creates draft package and assets
4. Outreach Agent drafts sequence and handoff
5. Executive Assistant summarizes opportunity for approval
6. Compliance/Coordination Agent logs and tracks the workflow

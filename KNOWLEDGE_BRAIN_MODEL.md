# KNOWLEDGE_BRAIN_MODEL.md

## Observed Marblism Pattern
Marblism exposes a shared "Brain" / knowledge area where AI employees can use:
- memories
- images
- webpages
- documents
- cloud storage imports

Observed knowledge examples included:
- coach/team pages
- style guides
- brand guideline docs
- organization-specific webpages

## Product Implications
athletics-ai-workforce should support a shared knowledge system with:
- document ingestion
- webpage ingestion
- image/media references
- structured memories
- connector imports from cloud storage later

## Knowledge Scopes
### Organization knowledge
- brand docs
- athletics department descriptions
- target customer definitions
- style guides
- schedules / policy docs

### Worker knowledge
- role-specific playbooks
- examples
- role-specific assets

### Personal knowledge
- private notes and preferences

## UX Recommendation
Knowledge should be presented as a searchable card/grid system with source type filters and ownership visibility.

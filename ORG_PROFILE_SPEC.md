# ORG_PROFILE_SPEC.md

## Purpose
Define the persistent organization profile that powers team generation, worker behavior, and onboarding.

## Core Fields
- organization name
- website
- industry / vertical
- organization description
- target customers / audiences
- country
- primary language
- tone of voice
- brand style notes
- goals / outcomes

## athletics-specific extensions
- school name
- athletic department description
- conference / division
- sponsorship priorities
- donor / alumni priorities
- recruiting priorities
- compliance sensitivity flags

## Why it matters
The org profile should influence:
- generated workforce recommendation
- worker guidelines defaults
- onboarding copy
- knowledge suggestions
- pricing/package recommendations

## UX Requirement
Org profile should be editable after onboarding and visibly tied to worker outputs.

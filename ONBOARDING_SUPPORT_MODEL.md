# ONBOARDING_SUPPORT_MODEL.md

## Observed Marblism Pattern
Marblism does not rely only on self-serve onboarding.
Observed support layer includes:
- welcome email
- live setup invitation
- webinar invite / power session
- direct booking links
- scheduled onboarding through calendar tool

## Product Implications
athletics-ai-workforce should support a hybrid onboarding model:
- self-serve setup for speed
- optional guided setup for higher conversion and retention

## Recommended Support Modes
### Self-serve
- guided onboarding wizard
- generated AI team
- embedded setup checklist

### Guided onboarding
- live setup session booking
- implementation support for higher-tier plans
- optional concierge migration/import help

## Why this matters
For a product selling AI workers, assisted onboarding builds trust and improves time-to-value.

# WORKFLOW_CATALOG.md

## MVP Workflow Catalog

### 1. Sponsor Attrition Recovery
**Trigger:** scheduled scan or manual request

**Flow:**
1. Sponsorship Intelligence Agent runs attrition analysis
2. Rank categories by revenue gap
3. Match alumni / decision-makers where applicable
4. Create opportunity task
5. Route to Proposal & Outreach Agent
6. Send approval request if proposal/outreach should proceed
7. Log all actions and outcomes

### 2. Alumni Sponsor Matching
**Trigger:** user request or category analysis result

**Flow:**
1. Sponsorship Intelligence Agent identifies target category/company set
2. CSOS alumni cross-reference is executed
3. Candidate matches are ranked
4. Results are packaged into a task and briefing
5. Next step routed to proposal/outreach or executive review

### 3. Proposal Draft Workflow
**Trigger:** approved opportunity or manual request

**Flow:**
1. Proposal & Outreach Agent creates proposal shell
2. Assets are added / package assembled
3. Proposal is reviewed internally
4. Approval gate triggered
5. If approved, proposal status advances and handoff occurs

### 4. Executive Briefing Workflow
**Trigger:** daily schedule, before meetings, or after significant workflow changes

**Flow:**
1. Executive Assistant collects open tasks, approvals, and major opportunities
2. Chief of Staff Agent synthesizes priorities
3. Summary delivered as briefing

### 5. Compliance & Coordination Workflow
**Trigger:** pending approvals, stale tasks, or missing data

**Flow:**
1. Compliance & Coordination Agent checks pending workflow states
2. Missing requirements are flagged
3. Reminders are issued
4. Escalation sent if deadlines pass
